# 微信小程序版网易云音乐 练手demo
[github](https://github.com/a758801405/my-music) 地址

## 技术架构
- 用的mpvue 初始化项目  
- UI框架用的是vantUI
- HTTP请求，用的[flyio](https://wendux.github.io/dist/#/doc/flyio/readme)
- css预处理器 用的sass
- 微信小程序[api](https://developers.weixin.qq.com/miniprogram/dev/api/)
